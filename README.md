# Java Student Utility App

## Brief Description
This is a console-based Java application built using a menu-driven system.  
The program performs different utility operations such as addition, checking even or odd numbers, printing numbers using loops, and displaying days of the week.

This project demonstrates understanding of:
- Java syntax and structure
- Variables and data types
- Operators
- Decision making (if/else, switch)
- Loops (for, do-while)
- Methods
- Scanner for user input
- Git & GitHub workflow

---

## How to Run the Program

1. Clone the repository:
git clone https://github.com/Timtech-O/java-week1-student-utility-app.git

2. Navigate into the project folder:
cd java-week1-student-utility-app

3. Compile the program:
javac src/StudentUtilityApp.java

4. Run the program:
java -cp src StudentUtilityApp
